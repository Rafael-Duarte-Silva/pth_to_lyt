# pth-to-lyt

**[English](#english)** | **[Português](#português)**

---

## English

Library to convert `.PTH` files from **Live for Speed (LFS)** into `.LYT` layouts.

The main goal of this library is to make it easier to create **reference layouts** from track *path nodes*, allowing you to build a precise base for **delta time calculation in races**.

With the generated `.LYT`, you can:

- Create reference lines along the track layout  
- Measure car progress relative to the ideal racing line  
- Calculate time differences between laps or players  
- Build telemetry and performance analysis systems  

Instead of manually creating a layout in the LFS editor, this library automates the process using the real track data.

### Installation

```bash
npm install pth-to-lyt
```

# Português

Biblioteca para converter arquivos `.PTH` do **Live for Speed ​​(LFS)** em layouts `.LYT`.

O principal objetivo desta biblioteca é facilitar a criação de **layouts de referência** a partir dos *path nodes* das pistas, permitindo montar uma base precisa para **cálculo de delta time em corridas**.

Com o `.LYT` gerado, é possível:

- Criar linhas de referência ao longo do traçado da pista
- Medir progresso do carro em relação ao caminho ideal
- Calcular diferenças de tempo entre voltas ou jogadores
- Construir sistemas de telemetria e análise de desempenho

Em vez de criar manualmente um layout no editor do LFS, esta lib automatiza o processo usando os dados reais da pista.
